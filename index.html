macOS: Unzip, then right-click the app → Open to get past Gatekeeper (the launcher isn't signed yet during alpha).
Windows: Single .exe — just run it. SmartScreen may flag the unsigned exe → click More info → Run anyway.
Linux: Single binary — make it executable first (chmod +x), then double-click or run from terminal.
Launcher won't auto-update?
If clicking Update Launcher fails or the launcher silently restarts on the old version, just download the latest one from the buttons above and replace your existing file. The launcher saves your login + game install path, so you won't lose anything.

Windows: a self-update log gets written to %TEMP%\pulsewave_launcher_update.log. Open that file (paste %TEMP% into Explorer) and send it to us if you want help debugging.
